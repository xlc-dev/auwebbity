<h1 align="center">Auwebbity</h1>

<p align="center" style="font-size: 1.2em; color: #586069;">
  A modern, browser-based multi-track audio editor.
</p>

---

Auwebbity is a powerful web-based audio editor that runs entirely in your browser. Built with Solid.js and WaveSurfer.js, it provides professional-grade audio editing capabilities without requiring any server-side processing.

## Features

- Multi-track audio editing with unlimited tracks
- Multiple waveform visualization modes (bars, line, spectrogram)
- Audio recording directly in the browser
- Import audio files in various formats
- Cut, copy, paste, delete, and other editing operations
- Audio effects: normalize, amplify, silence, reverse, fade in/out
- Project save/load functionality
- Undo/redo support
- Export to WAV, MP3, or OGG formats
- Per-track volume, mute, and solo controls
- Keyboard shortcuts for efficient editing
